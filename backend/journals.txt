# 找 RSS 的方法：
# 1. 打开期刊官网，找 RSS/订阅 图标
# 2. 在知网期刊页面，浏览器地址栏看是否有 RSS 链接
# 3. Google/Bing 搜索 "期刊名 RSS"
# 4. 维普期刊页面底部通常有 RSS 链接
